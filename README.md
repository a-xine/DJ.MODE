Installation & Setup

Clone the Repository
Bash
    git clone https://github.com/YOUR_USERNAME/DJ.MOD.git
    cd DJ.MOD
    
Install Dependencies
This will install Electron and any other required libraries.

Bash
    npm install
    
Run the App
Launch the music player in development mode.

Bash
  npm start
